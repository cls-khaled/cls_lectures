02- To by=e Sure Node Js Install Confirmation
- open Cmd Terminal 
- type Command  : node -v
-type command : npm -v



03- Install npm in the Project : create package.json file 
- open Cmd 
- type : npm init




04- Donwload Axios Library by Npm create package-lock.json file & node_module Folder 
download and save axios library in package.json file 

//----------- https://axios-http.com/docs/intro
- open cmd 
- type : npm install axios --save


05- link to axios.js from  node_modules Folder 

<script src="./node_modules/axios/dist/axios.js"></script>
